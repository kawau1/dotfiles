# dotfiles

## Tree Diagram

```
dotfiles/
├─ .zsh/
│  ├─ .p10k.zsh
│  ├─ .zshrc
│  └─ aliases.zsh (~/.oh-my-zsh/custom/)
├─ coding agent/
│  ├─ AGENTS.md
│  ├─ config.toml (~/.codex/)
│  └─ settings.json (~/.gemini/)
├─ .vimrc
├─ brew_setup.sh
├─ install.sh
├─ link.sh
└─ yt-dlp.conf (~/.config/yt-dlp/)


```

## Installetion

1. Clone this repository:

   ```bash
   git clone https://github.com/kawau1/dotfiles.git
   ```

2. Run the installation script:

   ```bash
   cd ./dotfiles
   chmod +x ./install.sh
   sh ./install.sh
   ```
